# Loglyzer – DNS & Zeek Log Analysis Tool

Loglyzer is a web-based log analysis system designed to parse and analyze DNS and Zeek logs to detect suspicious network activities. It helps in identifying potential threats by converting raw log data into meaningful insights through automated processing, rule-based detection, and visualization.

## 🔍 Overview

Modern networks generate massive amounts of log data, making manual analysis inefficient and time-consuming. Loglyzer simplifies this process by:

- Parsing DNS and Zeek logs
- Detecting abnormal patterns in network activity
- Generating alerts for suspicious behavior
- Visualizing insights through an interactive dashboard

This makes it a useful tool for network security monitoring and intrusion detection.

## ⚙️ Features
### - 📂 Log Parsing
  Extracts key information such as IP addresses, timestamps, DNS queries, and protocols.
### - 🚨 Attack Detection
  Identifies suspicious patterns like:
    - Repeated requests from same IP
    - Abnormal DNS query frequency
    - Suspicious domain access
### - 🔔 Alert Generation
  Generates alerts for potential threats in real time.
### - 📊 Visualization Dashboard
  Displays:
    - Top attacking IPs
    - Alerts and logs
    - Network activity statistics
